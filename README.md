# Edutech Solution Data Analytics Internship
## Task 1: Data Collection & Data Source Identification

### Part 1: Data Source Report
* **Dataset Name**: Electronic Product Pricing Data
* **Data Source**: Kaggle Public Repository
* **Data Type**: Secondary Data (collected by a third party)
* **Format**: CSV (Comma-Separated Values)

### Part 2: Interview Questions & Answers
* **Q1: What is the difference between Primary vs Secondary data?**
  * *Answer*: Primary data is collected firsthand by yourself. Secondary data is collected by someone else previously (like Kaggle datasets).
* **Q2: What are common data formats?**
  * *Answer*: Common formats include CSV, JSON, and SQL tables.
